## Deploy & Basic Usage

1.环境要求：

 - NodeJs >= v4.4.5
 - Redis v3.2.1
 - MongoDB v3.2.10

2.跑起服务：

> 启动 Redis 和 MongoDB (必须)

3.初始化站点：

```bash
./setup.sh
```

4.访问站点：

> http://localhost:3000
